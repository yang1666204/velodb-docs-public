## 説明

ポイントが有効なPOINT型の場合、対応するX座標値を返します。

## 構文

```sql
ST_X( <point>)
```
## Parameters

| Parameter   | Description       |
|------|----------|
| `<point>` | X座標を抽出する幾何オブジェクト。有効なPOINT型（2次元ポイント）である必要があり、X（経度）の範囲は[-180, 180]、Y（緯度）の範囲は[-90, 90]です。|

## 戻り値

幾何座標のX値、double精度浮動小数点数（Double）型。

- 入力が有効なPOINTオブジェクトの場合、ポイントのX座標（double精度浮動小数点数、Double）を返します。
- 入力がNULL、非POINT型オブジェクト、空のポイント（POINT EMPTY）、または無効なポイント（例：3次元ポイント）の場合、NULLを返します。

## Example


有効なポイントのX座標を抽出する

```sql
SELECT ST_X(ST_Point(24.7, 56.7));
```
```text
+----------------------------+
| st_x(st_point(24.7, 56.7)) |
+----------------------------+
|                       24.7 |
+----------------------------+
```
入力は空のポイントです（POINT EMPTY）

```sql
mysql> SELECT ST_X(ST_GeometryFromText("POINT EMPTY"));
+------------------------------------------+
| ST_X(ST_GeometryFromText("POINT EMPTY")) |
+------------------------------------------+
|                                     NULL |
+------------------------------------------+
```
入力は3Dポイントです（サポートされていません）

```sql

mysql> SELECT ST_X(ST_GeometryFromText("POINT (10 20 30)"));
+-----------------------------------------------+
| ST_X(ST_GeometryFromText("POINT (10 20 30)")) |
+-----------------------------------------------+
|                                          NULL |
+-----------------------------------------------+
```
入力がNULLです

```sql
mysql> SELECT ST_X(NULL);
+------------+
| ST_X(NULL) |
+------------+
|       NULL |
+------------+
```
経度が範囲外です

```sql
mysql> SELECT ST_X(ST_Point(244.7, 56.7));
+-----------------------------+
| ST_X(ST_Point(244.7, 56.7)) |
+-----------------------------+
|                        NULL |
+-----------------------------+
```
緯度が範囲外です

```sql
mysql> SELECT ST_X(ST_Point(44.7, 156.7));
+-----------------------------+
| ST_X(ST_Point(44.7, 156.7)) |
+-----------------------------+
|                        NULL |
+-----------------------------+
```

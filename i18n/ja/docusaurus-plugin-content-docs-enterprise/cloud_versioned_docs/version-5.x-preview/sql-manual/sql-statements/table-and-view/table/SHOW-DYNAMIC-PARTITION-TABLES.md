## 説明

このステートメントは、現在のデータベース内のすべての動的パーティションテーブルのステータスを表示するために使用されます。

## 構文：

```sql
SHOW DYNAMIC PARTITION TABLES [FROM <db_name>];
```
## 必須パラメータ
**1. `<db_name>`**

動的パーティションテーブルのステータスを表示する`DB`名を指定します。指定しない場合、デフォルトで現在の`DB`内のすべての動的パーティションテーブルのステータスが表示されます。

## 戻り値

| Column                 | DataType | Note                                                                                                                                                                                                                                                                                                                                                                    |
|------------------------|----------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| TableName              | varchar  | 現在のDBまたは指定されたDB内のテーブル名。                                                                                                                                                                                                                                                                                                                            |
| Enable                 | varchar  | テーブルの動的パーティションプロパティが有効かどうか                                                                                                                                                                                                                                                                                                                          |
| TimeUnit               | varchar  | 動的パーティションテーブルのパーティション粒度で、`HOUR`、`DAY`、`WEEK`、`MONTH`、`YEAR`が含まれます。                                                                                                                                                                                                                                                                             |
| Start                  | varchar  | 動的パーティションの開始オフセットで、負の数です。デフォルト値は-2147483648で、履歴パーティションが削除されないことを意味します。time_unit属性に応じて、現在の日（週/月）を基準にこのオフセットより前の範囲のパーティションが削除されます。                                                         |
| End                    | varchar  | 動的パーティションの終了オフセットで、正の数です。time_unit属性に応じて、現在の日（週/月）を基準に対応する範囲内のパーティションが事前に作成されます。                                                                                                                                                   |
| Prefix                 | varchar  | 動的に作成されるパーティション名のプレフィックス。                                                                                                                                                                                                                                                                                                                                   |
| Buckets                | varchar  | 動的に作成されるパーティションに対応するバケット数。                                                                                                                                                                                                                                                                                                                               |
| ReplicationNum         | varchar  | 動的に作成されるパーティションに対応するレプリカ数。指定しない場合、テーブル作成時に指定されたレプリカ数がデフォルトになります。                                                                                                                                                                                                                |
| ReplicaAllocation      | varchar  | 動的に作成されるパーティションに対応するレプリカ配布戦略。指定しない場合、テーブル作成時に指定されたレプリカ配布戦略がデフォルトになります。                                                                                                                                                                          |
| StartOf                | varchar  | 動的パーティショニングの各パーティション粒度の開始点。time_unitがWEEKの場合、このフィールドは各週の開始点を表し、値の範囲はMONDAYからSUNDAYです。time_unitがMONTHの場合、各月の開始日を表し、値の範囲は1日から28日です。time_unitがMONTHの場合、この値はデフォルトでNULLになります。 |
| LastUpdateTime         | datetime | 動的パーティションの最後の更新時間で、デフォルトではNULLです。                                                                                                                                                                                                                                                                                                                  |
| LastSchedulerTime      | datetime | 動的パーティションの最後のスケジューリング時間。                                                                                                                                                                                                                                                                                                                      |
| State                  | varchar  | 動的パーティションの状態。                                                                                                                                                                                                                                                                                                                                     |
| LastCreatePartitionMsg | varchar  | 動的パーティション追加スケジューリングの最後の実行からのエラーメッセージ。                                                                                                                                                                                                                                                                                                                                                    |
| LastDropPartitionMsg   | varchar  | 動的パーティション削除スケジューリングの最後の実行からのエラーメッセージ。                                                                                                                                                                                                                                                                                                                                                    |
| ReservedHistoryPeriods | varchar  | 動的パーティションによって保持される履歴パーティションのパーティション範囲で、動的パーティションテーブル内でどの履歴パーティションが自動削除されずに保持されるべきかを示します。                                                                                                                                                                                                                                                                                                                       |

## アクセス制御要件
1. パラメータdb_nameが指定されていない場合、現在のDB内のすべての動的パーティションテーブルのステータスが表示され、ユーザーがデフォルトで現在のDBの`SHOW_PRIV`権限を持っていると仮定されます。
2. パラメータdb_nameが指定されている場合、指定されたDB内のすべての動的パーティションテーブルのステータスが表示され、ユーザーはそのDBの`SHOW_PRIV`権限を持っている必要があります。

## 説明

`x`をeを底とする指数として返します。

## エイリアス

- DEXP

## 構文

```sql
EXP(<x>)
```
## Parameters

| Parameter | Description |
| -- | -- |
| `<x>` | 独立変数 |

## Return Value

double型の値を返します。特殊なケース：
- パラメータがNULLの場合、NULLを返します
- パラメータが非常に大きい場合、Infinityを返します

## Example

```sql
select exp(2);
```
```text
+------------------+
| exp(2.0)         |
+------------------+
| 7.38905609893065 |
+------------------+
```
```sql
select exp(3.4);
```
```text
+--------------------+
| exp(3.4)           |
+--------------------+
| 29.964100047397011 |
+--------------------+
```
```sql
select exp(1000000);
```
```text
+--------------+
| EXP(1000000) |
+--------------+
|     Infinity |
+--------------+
```

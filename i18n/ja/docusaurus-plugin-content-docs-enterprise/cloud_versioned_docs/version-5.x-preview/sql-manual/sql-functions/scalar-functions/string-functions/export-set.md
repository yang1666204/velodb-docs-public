## 説明

`EXPORT_SET`は、整数の各ビットを指定した文字列に変換し、それらを結果文字列として連結するために使用されます。`bits`内で1であるビットごとに、結果内の対応する位置に`on`文字列が表示され、0であるビットごとに`off`文字列が表示されます。ビットは右から左へ順序で（つまり、最下位ビットから最上位ビットへ）チェックされますが、左から右へ結果文字列に連結されます。ビットは`separator`で区切られます（デフォルトはカンマ`,`）。

表示されるビット数は`number_of_bits`パラメータで決定され、デフォルトは64ビットです。指定されていないか範囲外（64より大きい）の場合、自動的に64ビットとして扱われます。

この関数はMySQLの[EXPORT_SET function](https://dev.mysql.com/doc/refman/8.4/en/string-functions.html#function_export-set)と同様の動作をします。

## 構文

```sql
EXPORT_SET(bits, on, off[, separator[, number_of_bits]])
```
## パラメータ

| Parameter         | Description |
|-------------------|-------------|
| `bits`            | 変換に使用する整数型(LARGEINT)。値が-2^63未満の場合、-2^63として扱います。2^64 - 1より大きい場合、2^63 - 1として扱います。 |
| `on`              | ビットが1の時に表示される文字列 |
| `off`             | ビットが0の時に表示される文字列 |
| `separator`       | オプション、ビットを区切るために使用される文字列、デフォルトは',' |
| `number_of_bits`  | オプション、結果に表示するビット数、整数型(INT)をサポートします。デフォルト値は64で、範囲[0, 64]を超える場合はデフォルト値で自動的に処理されます。 |

## 戻り値

文字列を返します。`bits`の各ビットに対して、最下位ビットから最上位ビットまで、1の場合は`on`に置き換え、0の場合は`off`に置き換えて、順番に連結します。ビットは`separator`で区切られ、最終結果は`number_of_bits`ビットを表示します。

`number_of_bits`が範囲[-2^31, 2^31 - 1]を超えている場合、または関数内の任意のパラメータがNULLの場合、NULLを返します。

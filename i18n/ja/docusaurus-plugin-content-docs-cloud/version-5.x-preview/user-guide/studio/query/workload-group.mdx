# Workload Group Management
:::info
Workload Group ManagementはApache Doris 2.1.0 /VeloDB Cloud 4.0.0以上をサポートしています。
:::

## Workload Groupリスト
Workload Group Managementは、Workload Groupの迅速な作成、編集、および表示をサポートします。Workload Groupを使用することで、クラスタ内でのクエリとインポート負荷で使用されるCPU/メモリ/IOリソースの使用量を管理し、クラスタ内のクエリの最大同時実行数を制御できます。

<img className='pt-5' src={`/img/studio/workload.png`} alt="workload" width={1400} height={600}/>

Workload Groupリストの上にあるTableフィルタで、より多くの項目を表示できます。
<img className='pt-5' src={`/img/studio/workload-more.png`} alt="workload-more" width={1400} height={600}/>

## 新しいWorkload Groupの作成
新しいWorkload Groupインターフェースでは、パラメータのクエスチョンマークをクリックすると、パラメータの説明が表示されます。

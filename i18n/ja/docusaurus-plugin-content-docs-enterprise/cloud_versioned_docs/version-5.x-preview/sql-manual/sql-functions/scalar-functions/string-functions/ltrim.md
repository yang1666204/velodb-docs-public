## 説明

LTRIM関数は、文字列の左側（先頭）から連続する空白文字または指定した文字セットを削除します。この関数は文字列の左端からスキャンし、対象文字セットに含まれない文字に遭遇するまで、連続するすべての対象文字を削除します。

## 構文

```sql
LTRIM(<str> [, <trim_chars>])
```
## パラメータ

| パラメータ      | 説明                                                                                                                                                                                                                                                                           |
|----------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `<str>`        | 左トリムされる元の文字列。型: VARCHAR                                                                                                                                                                                                                                  |
| `<trim_chars>` | 省略可能なパラメータで、削除する文字のセットを指定します。提供されない場合は、デフォルトで空白文字が削除されます。型: VARCHAR |

## 戻り値

VARCHAR型を返し、左側から指定された文字を削除した後の文字列を表します。

トリミングルール:
- 文字列の左側（先頭）からのみ文字を削除します
- trim_charsに含まれる連続するすべての文字を削除します
- trim_charsに含まれない文字が見つかると削除を停止します
- trim_charsが指定されない場合は、デフォルトで空白文字（スペース、タブ、改行など）を削除します

特殊なケース:
- いずれかのパラメータがNULLの場合、NULLを返します
- strが空文字列の場合、空文字列を返します
- trim_charsが空文字列の場合、元の文字列を返します
- 文字列全体がtrim_charsの文字で構成されている場合、空文字列を返します

## 例

1. 左側の空白を削除

```sql
SELECT LTRIM('   ab d');
```
```text
+-------------------+
| LTRIM('   ab d')  |
+-------------------+
| ab d              |
+-------------------+
```
2. 指定された文字を削除する

```sql
SELECT LTRIM('ababccaab', 'ab');
```
```text
+----------------------------+
| LTRIM('ababccaab', 'ab')   |
+----------------------------+
| ccaab                      |
+----------------------------+
```
3. NULL値の処理

```sql
SELECT LTRIM(NULL), LTRIM('test', NULL);
```
```text
+-------------+---------------------+
| LTRIM(NULL) | LTRIM('test', NULL) |
+-------------+---------------------+
| NULL        | NULL                |
+-------------+---------------------+
```

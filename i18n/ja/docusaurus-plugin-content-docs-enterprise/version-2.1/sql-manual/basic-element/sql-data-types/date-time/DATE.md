# DATE

## DATE

DATE

### デスクリプション
#### Syntax
date
Date型で、現在の値の範囲は['0000-01-01','9999-12-31']であり、デフォルトの出力形式は'yyyy-MM-dd'です。

### example

```sql
SELECT DATE('2003-12-31 01:02:03');
+-----------------------------+
| DATE('2003-12-31 01:02:03') |
+-----------------------------+
| 2003-12-31                  |
+-----------------------------+
```

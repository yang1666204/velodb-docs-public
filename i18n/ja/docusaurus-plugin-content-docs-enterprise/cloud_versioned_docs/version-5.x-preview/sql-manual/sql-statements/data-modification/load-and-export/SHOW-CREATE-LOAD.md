## Description

このステートメントは、importジョブの作成ステートメントを表示するために使用されます。

## Syntax

```sql
SHOW CREATE LOAD FOR <load_name>;
```
## 必須パラメータ

**`<load_name>`**

> ルーチンインポートジョブの名前。

## アクセス制御要件

このSQLコマンドを実行するユーザーは、少なくとも以下の権限を持つ必要があります：

| 権限 | オブジェクト | 備考 |
| :---------------- | :------------- | :---------------------------- |
| ADMIN/NODE_PRIV | Database | クラスタ管理者権限が必要です。 |

## 戻り値

指定されたインポートジョブの作成ステートメントを返します。

## 例

- デフォルトデータベース内の指定されたインポートジョブの作成ステートメントを表示します。

```sql
SHOW CREATE LOAD for test_load
```

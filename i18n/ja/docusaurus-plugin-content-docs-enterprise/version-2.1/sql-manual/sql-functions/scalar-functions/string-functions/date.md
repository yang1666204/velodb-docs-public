# DATE

## 説明

datetimeから日付を抽出します。

## 構文

```sql
DATE(<datetime>)
```
## パラメータ

| Parameter | デスクリプション |
| -- | -- |
| `<datetime>` | 有効な日付式 |

## 戻り値

datetimeの日付部分を返します。

## 例

```sql
select date('2010-12-02 19:28:30');
```
```text
+----------------------------------------------------+
| date(cast('2010-12-02 19:28:30' as DATETIMEV2(0))) |
+----------------------------------------------------+
| 2010-12-02                                         |
+----------------------------------------------------+
```

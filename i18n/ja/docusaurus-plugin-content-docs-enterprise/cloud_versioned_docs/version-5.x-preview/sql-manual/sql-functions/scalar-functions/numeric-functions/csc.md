## 説明

xのコセカントを返します。ここでxはラジアン単位の値であり、入力と出力はdoubleのみサポートされています。入力のnull値はnull値を返します。

## 構文

```sql
CSC(<x>)
```
## Parameters

| Parameter | Description |
| -- | -- |
| `<x>` | 余割を計算する値 |

## Return Value

xの余割を表すDouble型の値を返します。

## Special Cases
- `x`がNaNの場合、NaNを返します
- `x`が正または負の無限大の場合、NaNを返します
- `x`がNULLの場合、NULLを返します

## Example

```sql
select csc(1),csc(2),csc(1000);
```
```text
+--------------------+--------------------+------------------+
| csc(1)             | csc(2)             | csc(1000)        |
+--------------------+--------------------+------------------+
| 1.1883951057781212 | 1.0997501702946164 | 1.20936599707935 |
+--------------------+--------------------+------------------+
```
null値を入力します。

```sql
select csc(null);
```
```text
+--------------------+
| csc(null)          |
+--------------------+
|      NULL          |
+--------------------+
```
```sql
select csc(cast('nan' as double));
```
```text
+---------------------------+
| csc(cast('nan' AS DOUBLE))|
+---------------------------+
| NaN                       |
+---------------------------+
```
```sql
select csc(cast('inf' as double));
```
```text
+---------------------------+
| csc(cast('inf' AS DOUBLE))|
+---------------------------+
| NaN                       |
+---------------------------+
```
```sql
select csc(cast('-inf' as double));
```
```text
+----------------------------+
| csc(cast('-inf' AS DOUBLE))|
+----------------------------+
| NaN                        |
+----------------------------+
```

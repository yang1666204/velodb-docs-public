## 概要

MASK関数は機密情報を保護するためにデータをマスクする機能であり、データ匿名化のシナリオで一般的に使用されます。デフォルトの動作では、入力文字列内の大文字を`X`に、小文字を`x`に、数字を`n`に変換します。

## 構文

```sql
MASK(<str> [, <upper> [, <lower> [, <number> ]]])
```
## パラメータ

| Parameter  | デスクリプション                                                                                                                                                                                                       |
|------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `<str>`    | マスクする必要がある文字列                                                                                                                                                                                     |
| `<upper>`  | オプションパラメータ。デフォルトで大文字を`X`に置き換える。文字列が入力された場合、最初の文字が使用され、非ASCII文字が入力された場合、最初のバイトが使用される |
| `<lower>`  | オプションパラメータ。デフォルトで小文字を`x`に置き換える。文字列が入力された場合、最初の文字が使用され、非ASCII文字が入力された場合、最初のバイトが使用される |
| `<number>` | オプションパラメータ。デフォルトで数字を`n`に置き換える。文字列が入力された場合、最初の文字が使用され、非ASCII文字が入力された場合、最初のバイトが使用される   |

## Return Value

大文字、小文字、数字をマスクした後の文字列を返す。特殊なケース：

- いずれかのParameterがNULLの場合、NULLが返される。
- 英字でも数字でもない文字はマスクされない

## Examples

```sql
select mask('abc123EFG');
```
```text
+-------------------+
| mask('abc123EFG') |
+-------------------+
| xxxnnnXXX         |
+-------------------+
```
```sql
select mask(null);
```
```text
+------------+
| mask(NULL) |
+------------+
| NULL       |
+------------+
```
```sql
select mask('abc123EFG', '*', '#', '$');
```
```text
+----------------------------------+
| mask('abc123EFG', '*', '#', '$') |
+----------------------------------+
| ###$$$***                        |
+----------------------------------+
```

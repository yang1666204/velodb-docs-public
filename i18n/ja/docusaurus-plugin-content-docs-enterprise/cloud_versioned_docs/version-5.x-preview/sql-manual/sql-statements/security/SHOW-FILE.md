## Description

このステートメントは、データベース内に作成されたファイルを表示するために使用されます。

## Syntax

```sql
SHOW FILE { [ FROM | IN ] <database_name>}
```
## オプションパラメータ

**1. `<database_name>`**

> ファイルが属するデータベースを指定します。指定されない場合は、現在のセッションのデータベースを使用します。

## 戻り値

| カラム      | 説明                     |
|:------------|:-------------------------|
| `FileId`    | グローバルに一意なファイル識別子 |
| `DbName`    | 所有者データベースの名前 |
| `Catalog`   | ユーザー定義の分類       |
| `FileName`  | ファイルの名前           |
| `FileSize`  | ファイルサイズ（バイト） |
| `IsContent` | コンテンツ存在フラグ     |
| `MD5`       | ファイルのMD5チェックサム |

## アクセス制御要件

このSQLコマンドを実行するユーザーは、以下の最小権限を持つ必要があります：

| 権限         | オブジェクト | 備考                                                                     |
|:-------------|:-------------|:-------------------------------------------------------------------------|
| `ADMIN_PRIV` | User / Role  | このコマンドを実行するには、ファイルが存在するデータベースへのアクセス権限をユーザーまたはロールが持つ必要があります |

## 例

- 現在のセッションデータベース内のアップロード済みファイルを表示

   ```sql
   SHOW FILE;
   ```
   ```text
   +-------+--------+----------+--------------------------+----------+-----------+----------------------------------+
   | Id    | DbName | Catalog  | FileName                 | FileSize | IsContent | MD5                              |
   +-------+--------+----------+--------------------------+----------+-----------+----------------------------------+
   | 12006 | testdb | doris_be | doris_be_metadata_layout | 89349    | true      | 9a3f68160b4106b0e923a0aa2fc05599 |
   +-------+--------+----------+--------------------------+----------+-----------+----------------------------------+
   ```
- 指定されたデータベース内のアップロードされたファイルを表示する

   ```sql
   SHOW FILE FROM example_db;
   ```
   ```text
   +-------+------------+----------+--------------------------+----------+-----------+----------------------------------+
   | Id    | DbName     | Catalog  | FileName                 | FileSize | IsContent | MD5                              |
   +-------+------------+----------+--------------------------+----------+-----------+----------------------------------+
   | 12007 | example_db | doris_fe | doris_fe_metadata_layout | 569373   | true      | 10385505d3c0d03f085fea6f8d51adfa |
   | 12008 | example_db | doris_be | doris_be_metadata_layout | 89349    | true      | 9a3f68160b4106b0e923a0aa2fc05599 |
   +-------+------------+----------+--------------------------+----------+-----------+----------------------------------+
   ```

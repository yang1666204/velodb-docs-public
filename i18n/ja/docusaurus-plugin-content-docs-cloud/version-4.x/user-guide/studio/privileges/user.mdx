# User

接続上のユーザーを表示し、ユーザーの追加、変更、削除、ユーザーのパスワード変更、ユーザーの認可を含むユーザー情報を管理できます。

:::warning
Admin権限を持つユーザーのみが他のユーザーを追加および変更できます。
:::

<br />

<img src={`/img/studio/user.png`} alt="role" width={1400} height={600}/>

## 新しいユーザーを追加

このページで新しいユーザーを作成できます。Studioはセキュリティを強化するためユーザーにパスワードを必須としています。

<br />

<img src={`/img/studio/add-user.png`} alt="role" width={1400} height={600}/>


## ユーザーを編集

ユーザーを編集する際、ユーザー名、ロール、ホストを含むユーザーの情報を変更できます。
<img className='pt-5' src={`/img/studio/edit-user.png`} alt="edit-user" width={1400} height={600}/>

## パスワードを更新

ユーザーのパスワードを更新できます

<img className='pt-5' src={`/img/studio/update-password.png`} alt="update-password" width={1400} height={600}/>

## ユーザーを削除

ユーザーを削除する際、確認ボックスが表示されます。削除を確認した後、ユーザーが削除されます。
<img className='pt-5' src={`/img/studio/delete-user.png`} alt="delete-user" width={1400} height={600}/>

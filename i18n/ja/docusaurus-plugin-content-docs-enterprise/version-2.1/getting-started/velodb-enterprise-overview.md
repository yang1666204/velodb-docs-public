# はじめに

VeloDB Enterprise は Apache Doris を基盤とする商用製品です。オンプレミス、VM、または K8s での使用が推奨されており、コミュニティ版と比較してより優れた運用・開発体験を実現し、セキュリティと安定性への強力なサポートを提供します。

## 長期コアバージョン

オープンソースのコアバージョンは急速に反復され、ユーザーに新機能や新しい機能の体験を提供しますが、いくらかの安定性を犠牲にします。オープンソース版とは対照的に、エンタープライズ版は長期サポート付きの安定した配布を提供し、本番環境での安定性を確保するために重要なバグの専用修正を日次で提供します。

* **安定性** : コミュニティによる広範な使用と厳格なテストを通じて厳密にテストされた安定したコア。
* **Long-Term Support (LTS)** : 12-24ヶ月のサポートを持つLTSバージョンを提供し、定期的にバグ修正版をリリース。
* **互換性** : Doris コミュニティエディションと**完全に互換性があり**、**シームレスな移行が可能**。コミュニティの更新後、VeloDB も対応する配布を提供し、最新の機能と最適化との整合性を確保します。

## 運用・開発ツール

直感的なインターフェイスを通じてユーザーにシンプルで使いやすいビジュアル運用体験を提供する、ワンストップのデータベースクラスタ管理ツールを提供します。これにより運用効率が大幅に向上し、エラーや中断が減少し、サービス品質と安定性が向上します。

* **クラスタのデプロイとテイクオーバー** ：オンプレミス、VM、K8s のいずれであっても、VeloDB Manager は Doris クラスタを簡単にデプロイし、テイクオーバーできます。複雑なコマンドライン操作は不要で、すべて直感的なインターフェイスで実行できます。
* **柔軟なクラスタ管理** ：必要に応じてクラスタのスケーリング、再起動、アップグレード、パラメータ設定操作を実行。これによりビジネスの柔軟性とシステム可用性が大幅に向上します。
* **監視とアラート** ：VeloDB Manager には事前定義された監視メトリクスセットが付属し、カスタマイズ可能なアラート通知をサポートします。ユーザーはニーズに応じてアラートしきい値と通知方法を設定し、クラスタ運用に関する情報を適時に受信し、早期警告と積極的な問題解決を可能にします。
* **便利な WebUI** ：各クラスタに WebUI エントランスを提供し、ユーザーが簡単にクエリ実行、データ表示、権限管理を行えます。

## 有償技術サポートサービス

Apache Doris の創設チームによって構築された VeloDB は、本番環境での Doris 使用に関連するあらゆる懸念に対処する、迅速で業界をリードする技術サポートを提供する能力があります。VeloDB Enterprise は、お客様の特定のニーズに合わせたサービスレベル契約（SLA）に基づく専門的な有償技術サポートを提供します。このサポートには以下が含まれます：

* **リスク軽減** ：定期的な検査を実施し、潜在的なシステムリスクを迅速に特定・排除。
* **問題解決** ：24時間365日の専用サポートを保証する厳格なサービスレベル契約。重要なバグの専用修正を日次更新で提供。
* **システム最適化** ：製品使用トレーニング、業界のベストプラクティスの共有、お客様との協力的な最適化努力を通じて、システムのパフォーマンスとコスト効率を向上。

## 説明

`x`の双曲線逆正接を返します。`x`が`-1`から`1`の範囲内にない場合（`-1`と`1`を除く）は`NULL`を返します。

## 構文

```sql
ATANH(<x>)
```
## Parameters

| Parameter | Description |  
| -- | -- |  
| `<x>` | 双曲線逆正接を計算する値 |  

## Return Value  

パラメータ`x`の双曲線逆正接値。

## Special Cases

- `x`が0の場合、0を返す
- `x`が1または-1の場合、`NULL`を返す
- `x`が範囲(-1, 1)の外側にある場合、`NULL`を返す
- `x`がNaNの場合、NaNを返す
- `x`が正または負の無限大の場合、`NULL`を返す
- `x`がNULLの場合、NULLを返す

## Examples

```sql
select atanh(0.0);
```
```text
+------------+
| atanh(0.0) |
+------------+
|          0 |
+------------+
```
```sql
select atanh(-1.0);
```
```text
+-------------+
| atanh(-1.0) |
+-------------+
|        NULL |
+-------------+
```
```sql
select atanh(1.0);
```
```text
+------------+
| atanh(1.0) |
+------------+
|       NULL |
+------------+
```
```sql
select atanh(0.5);
```
```text
+--------------------+
| atanh(0.5)         |
+--------------------+
| 0.5493061443340548 |
+--------------------+
```
```sql
select atanh(cast('nan' as double));
```

```text
+---------------------------+
```
```sql
select atanh(cast('inf' as double));
```
```text
+---------------------------+
| atanh(cast('inf' AS DOUBLE)) |
+---------------------------+
| NULL                      |
+---------------------------+
```

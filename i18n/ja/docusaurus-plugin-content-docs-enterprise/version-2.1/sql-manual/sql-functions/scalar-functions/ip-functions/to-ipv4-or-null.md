## 説明
この関数は、IPv4アドレスの文字列形式を受け取り、IPv4型の値を返すipv4_string_to_num関数と同様の機能を持ちます。戻り値はipv4_string_to_numが返す値とバイナリ的に等しくなります。

## 構文

```sql
TO_IPV4_OR_NULL(<ipv4_str>)
```
## パラメータ
| Parameter | デスクリプション                                      |
|-----------|--------------------------------------------------|
| `<ipv4_str>`      | String型のIPv4アドレス |


## Return Value
IPv4型の値を返します。
- IPv4アドレスの形式が無効な場合、NULLを返します。


## Example

```sql
SELECT to_ipv4_or_null('255.255.255.255'), to_ipv4_or_null('.'), to_ipv4_or_null(NULL);
```
```text
+------------------------------------+----------------------+-----------------------+
| to_ipv4_or_null('255.255.255.255') | to_ipv4_or_null('.') | to_ipv4_or_null(NULL) |
+------------------------------------+----------------------+-----------------------+
| 255.255.255.255                    | NULL                 | NULL                  |
+------------------------------------+----------------------+-----------------------+
```

# Integration Overview

VeloDB integrations are categorized into **BI, Lakehouse, Observability, SQL client, Data Source, Data Ingestion and Data Processing** categories.

This list of VeloDB / Apache Doris integrations is continuously being updated and is not yet complete. We welcome any contributions of relevant VeloDB / Apache Doris integrations to help expand it. [Contact Us](mailto:contact@velodb.io) to update the integration list.

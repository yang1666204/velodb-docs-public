# 統合概要

VeloDBの統合は**BI、レイクハウス、監視、SQLクライアント、データソース、データ取り込み、データ処理**のカテゴリに分類されます。

このVeloDB / Apache Doris統合のリストは継続的に更新されており、まだ完全ではありません。関連するVeloDB / Apache Doris統合の貢献を歓迎し、拡充にご協力いただけます。統合リストの更新については[お問い合わせ](mailto:contact@velodb.io)ください。

# Smartbi

## Smartbi 紹介

Smartbiは、Oracle、SQL Server、MySQL、Dorisを含む様々なデータソースに接続できるソフトウェアサービスとアプリケーションコネクタの集合体であり、ユーザーが簡単にデータを統合・クレンジングできるようにします。Smartbiのデータモデリング機能により、ユーザーは複雑なリレーショナルモデルを作成するだけでなく、データ分析式を記述し、データ関係を確立することができ、高度なデータ分析と可視化の基盤を築きます。SmartBIは、様々なタイプのチャート、地理的マップ、インタラクティブダッシュボード、カスタマイズ可能な可視化ツールを含む、多様な可視化オプションを提供します。これらの強力なツールは、ユーザーがデータをより直感的かつ包括的に理解し、提示することを支援し、データ分析の効果と効率を向上させます。

## 前提条件

https://www.smartbi.info/download にアクセスしてSmartbiをダウンロードし、インストールできます。

## データ接続とアプリケーション

1. Smartbiにログインし、Data Connectionsをクリックします。

   ![main page](/images/cloud/integration/bi/bi-smartbi-en-1.png)

2. data connectionsをクリックした後、接続するデータベースのリストが表示されるので、Dorisデータベースを選択します。

   ![velodb](/images/cloud/integration/bi/bi-smartbi-en-2.png)

3. データベースを選択した後、Dorisデータベースの接続情報を入力します。

   ![data source connection](/images/cloud/integration/bi/bi-smartbi-en-3.png)

4. 情報が正しく、ネットワークが接続されていれば、Testに合格したことを示すメッセージが表示されます。

   ![test passed](/images/cloud/integration/bi/bi-smartbi-en-4.png)

5. 接続が成功した後、データ分析モジュールSelf-service dashboardで必要なレポート情報をカスタマイズし、設定できます。

   ![data analysis](/images/cloud/integration/bi/bi-smartbi-en-5.png)

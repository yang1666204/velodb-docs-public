# Config Dir

FEとBEの設定ファイルディレクトリは`conf/`です。このディレクトリは、デフォルトのfe.conf、be.confおよびその他のファイルを格納するほか、共通の設定ファイル格納ディレクトリとしても使用されます。

ユーザーは設定ファイルをここに格納でき、システムが自動的に読み込みます。

:::tip Tips
この機能はApache Doris 1.2バージョン以降でサポートされています
:::

## hdfs-site.xmlとhive-site.xml

Dorisの一部の機能では、HDFS上のデータにアクセスしたり、Hive metastoreにアクセスする必要があります。

機能の対応するステートメントに、様々なHDFS/Hiveパラメータを手動で記入することができます。

しかし、これらのパラメータは非常に多く、すべてを手動で記入するのは非常に面倒です。

そのため、ユーザーはHDFSまたはHiveの設定ファイルhdfs-site.xml/hive-site.xmlを`conf/`ディレクトリに直接配置できます。Dorisはこれらの設定ファイルを自動的に読み込みます。

ユーザーがコマンドで記入した設定は、設定ファイル内の設定項目を上書きします。

このようにして、ユーザーは少量の設定を記入するだけで、HDFS/Hiveへのアクセスを完了できます。

# バックアップとリストア

VeloDB Cloudは、定期的または一回限りのオペレーションとして、データベースをオブジェクトストレージにバックアップすることをサポートし、指定されたバックアップセットに基づいてユーザーがデータを迅速にリストアできるようにし、データの高可用性を包括的に保証します。

## バックアップ

### バックアッププランの作成

左側のナビゲーションバーで**Backup**をクリックし、バックアップページで**Create Backup Plan**をクリックします。必要に応じて、定期バックアップまたは一回限りのバックアップを選択できます。定期バックアップと一回限りのバックアップは相互排他的な関係にあります。バックアッププランを更新すると、元のバックアッププランが上書きされます。

定期バックアップを選択した場合、有効にするかどうか、バックアップ実行サイクル、開始時刻、バックアップオブジェクト、保持日数、およびバックアップに使用するクラスターを選択し、これらの設定を保存して有効にする必要があります。

![backup-periodic](/images/cloud/backup-periodic.png)

一回限りのバックアップを選択した場合、開始時刻、バックアップオブジェクト、保持日数、およびバックアップに使用するクラスターを選択する必要があります。同様に、これらの設定を保存して有効にする必要があります。

![backup-one-time](/images/cloud/backup-one-time.png)

| パラメーター | 説明 |
| ---------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Backup Every                 | 月曜日から日曜日まで複数選択が可能で、最低1日、最大7日です。                                                        |
| Start Time                      | バックアップタスクの開始時刻。                                                                                                                        |
| Backup Objects                         | 内部カタログ：データベース；外部カタログ：DDLのみをバックアップし、データはバックアップしません。                                                                                          |
| Backup Retention Days                  | バックアップセットの保持日数を設定し、保持日数を超えたバックアップセットはクリアされます。                                                       |
| Backup Cluster             | バックアップ処理は計算リソースを消費します。複数のクラスターがある場合、バックアップ操作に使用するクラスターを指定する必要があります。 |

### バックアップタスクの表示

VeloDB Cloudは、設定したプランに従って自動的にバックアップタスクを実行します。**Backup Tasks**リストで、バックアップステータス、保持日数、データサイズ、バックアップの開始時刻と完了時刻を含むすべてのバックアップタスクを表示します。

![backup-list](/images/cloud/backup-list.png)

操作列の**View Details**をクリックして、バックアップタスク実行の詳細情報を取得します。

![backup-detail](/images/cloud/backup-detail.png)

## リストア

バックアップタスクのリストで対象のバックアップセットが配置されている行を選択し、操作列の**Restore**をクリックして、リストアタスクの対象ウェアハウスとクラスターを指定し、バックアップをリストアできます。

![backup-restore](/images/cloud/backup-restore.png)

リストアタスクは**Restore Tasks**リストに表示され、タスクステータス、データサイズ、開始時刻と完了時刻などの詳細情報を表示できます。

![restore-list](/images/cloud/restore-list.png)

操作列の**View Details**をクリックして、リストアタスク実行の詳細情報を取得します。

![restore-detail](/images/cloud/restore-detail.png)

# Apache Doris Documentation

VeloDB Enterprise 2.1 is a stable distribution based on Apache Doris 2.1, providing long-term support. For detailed documentation on the specific kernel, please refer to [Apache Doris 2.1](https://doris.apache.org/docs/2.1/gettingStarted/what-is-apache-doris).

VeloDB Enterprise 3.0 is a stable distribution based on Apache Doris 3.0, providing long-term support. For detailed documentation on the specific kernel, please refer to [Apache Doris 3.0](https://doris.apache.org/docs/3.x/gettingStarted/what-is-apache-doris).

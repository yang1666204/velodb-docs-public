# Release 注釈

:::info
VeloDB Studio Server版とDesktop版は共通のバージョン更新ログを使用していますが、Server版の一部機能はリリース日より早くまたは遅く更新される場合があります。
:::

## v1.1.0
リリース日: 2025年4月29日

VeloDB Studio v1.1.0がリリースされました！このバージョンでは、Windows版とMac Intel chip版をリリースし、SQLコンソールに大幅な更新を行い、さらにいくつかのバグ修正と機能最適化を行いました。

以下が具体的な更新内容です：

### 新機能

- SQLコンソールが複数行SQLクエリをサポート

- ProfileがオリジナルのText形式での表示をサポート

- REFRESH CATALOGをサポート

- 一部ユーザーのログインを制限するホワイトリストをサポート（Server版、Cloud BYOCリポジトリで実行）

- ユーザー/ロール管理にCOMMENTを追加

- SQLクエリ結果の複製をサポート...

### 最適化
- SQLクエリ結果が大容量データでのパフォーマンスを最適化

- StudioがMSサービスに依存しないように変更（Server版、Cloud SaaSリポジトリで実行）


### バグ修正

- 中国語検索が利用できない検索および分析の問題

- Load Management 0.0001 mem_limitエラー

- SQLコンソールでの部分ビュー表示エラー


## v1.0.0

リリース日: 2025年3月27日

VeloDB Studio Desktopの初回バージョンがリリースされました！これは重要なマイルストーンであり、このバージョンを皆様と共有できることを嬉しく思います。

このバージョンでは、以下の機能をお届けします：

- Apache Doris用に最適化されたSQLコンソール

- Apache Dorisログ機能の検索分析を提供

- ビジュアルセッション管理とクエリ監査

- ビジュアルProfile分析

- 権限管理

- データパネル

- ビジュアルインポートタスク表示

- マルチ接続とSSHトンネリングをサポート（Desktop版のみ）

- Dark /Light Theme（Server版は近日サポート予定）

このバージョンをお楽しみいただき、このバージョンを使用してフィードバックをお寄せいただき、改善にご協力いただければ幸いです。

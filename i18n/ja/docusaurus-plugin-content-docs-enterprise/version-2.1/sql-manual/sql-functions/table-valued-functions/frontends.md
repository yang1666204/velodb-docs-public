## Description

table関数はフロントエンドの一時テーブルを生成し、現在のdorisクラスタ内のFEノード情報を表示することができます。

## Syntax

```sql
FRONTENDS()
```
## アクセス制御要件

| Privilege  | Object | Notes |
| :--------- |:-------|:------|
| ADMIN_PRIV | global |       |

## 戻り値
| Field                    | Description                                                                                             |
|--------------------------|---------------------------------------------------------------------------------------------------------|
| **Name**                 | frontendノードの一意な名前。                                                                   |
| **Host**                 | frontendノードのIPアドレスまたはホスト名。                                                        |
| **EditLogPort**          | edit log通信に使用されるポート。                                                               |
| **HttpPort**             | frontendノードのHTTPポート。                                                                    |
| **QueryPort**            | frontendノードでのクエリ実行に使用されるポート。                                                 |
| **RpcPort**              | RPC通信に使用されるポート。                                                                    |
| **ArrowFlightSqlPort**   | Arrow Flight SQLポート（高性能なデータ転送のためのApache Arrowとの統合に使用される）。 |
| **Role**                 | frontendノードの役割（例：`FOLLOWER`）。                                                       |
| **IsMaster**             | ノードがマスターノードかどうかを示す（true/false）。                                               |
| **ClusterId**            | frontendノードが属するクラスターの識別子。                                      |
| **Join**                 | frontendノードがクラスターの一部かどうかを示す（true/false）。                                     |
| **Alive**                | frontendノードが生きているかどうかを示す（true/false）。                                                   |
| **ReplayedJournalId**    | frontendノードによって再生された最後のjournal ID。                                             |
| **LastStartTime**        | frontendノードの最後の開始時刻のタイムスタンプ。                                              |
| **LastHeartbeat**        | frontendノードから受信した最後のハートビートのタイムスタンプ。                                   |
| **IsHelper**             | frontendノードがヘルパーノードかどうかを示す（true/false）。                                      |
| **ErrMsg**               | frontendノードから報告されたエラーメッセージ。                                                       |
| **Version**              | frontendノードのバージョン。                                                                       |

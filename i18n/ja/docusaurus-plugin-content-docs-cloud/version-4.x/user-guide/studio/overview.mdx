# VeloDB Studio の紹介

VeloDB Studio は、Apache Doris およびその互換データベース向けに設計された GUI ツールで、データ開発と管理を簡素化します。

VeloDB Studio には Server と Desktop の 2 つのバージョンがあります：

- Server バージョンは、VeloDB Cloud と Enterprise でエンタープライズレベルのユーザーサービスを提供するために組み込まれています。

- Desktop バージョンは、コンピュータに直接インストールできるデスクトップアプリケーションで、Mac、Windows、Linux（将来対応予定）をサポートしています。

## 主要機能

### SQL Editor

Apache Doris 専用に設計された SQL エディタは、SQL 構文ハイライト、自動補完、フォーマットなどの機能をサポートし、SQL 記述効率を向上させます。

### ログ検索とビジュアル分析

ログ検索と可視化機能を提供し、Apache Doris と Studio のログ検索機能を組み合わせて、Elastic Search と Kibana Discover に代わるログストレージ、クエリ、可視化を実現し、10 倍のコスト削減とより効率的なデータ分析を達成できます。

{/* ### 会話管理

支持会话的监控和控制，用户可以查看当前活跃的会话，并根据需要进行管理，优化资源使用。 */}

### Query Audit

Query Audit は、Doris で実行されたクエリ履歴を監査・分析するために使用されます。スロークエリをフィルタリングしたり、ユーザー、ホスト、SQL 文などを通じてフィルタリングして監査要件を満たすことができます。

### 権限管理

Apache Doris のユーザー権限をビジュアルに管理し、データベースデータと操作のセキュリティを確保し、エンタープライズレベルアプリケーションのニーズを満たします。

### 複数接続と SSH トンネル（Desktop バージョンのみ）

マルチデータベース接続をサポートし、SSH トンネリング機能を提供することで、ユーザーがセキュアなチャネルを通じて Doris データベースをリモート管理することを容易にし、ネットワーク環境全体での互換性を向上させます。

## VeloDB Studio Server バージョン

:::info
Server バージョンは Chrome 90 以上のブラウザをサポートしており、最新版のブラウザの使用を推奨します。
:::

VeloDB Studio Server バージョンは VeloDB Cloud と Enterprise に組み込まれており、Web 形式でエンタープライズユーザーに提供されます。

### VeloDB Studio Server Edition の特徴

**1. 深い統合**: Server バージョンは VeloDB Cloud と Enterprise に深く統合されており、異なる Manager バージョンに応じて異なる適応機能を持っています。

**2. ネットワーク分離**: Enterprise Studio はエンタープライズレベルのネットワーク環境に配置され、VeloDB Cloud バージョンの Studio は VPC に配置されて、セキュアなネットワーク環境を提供します。

**3. より高い品質と安定性**: Server バージョンは安定性により重点を置き、新機能に対してより厳しい品質要件を持っています。

**4. セキュリティアップデート**: Server バージョンは、より迅速なセキュリティアップデートと脆弱性対応を提供し、脆弱性とセキュリティ問題を個別にアップデートして配信します。

**5. エンタープライズレベルサポート**: チームは専門的な技術サポートとより迅速な機能リクエストを提供し、Server バージョンの問題は常に最優先で対応されます。

**6. チーム協業**: Server バージョンはチーム協業により適しており、同じアクセスアドレスで複数のユーザーが Studio を共有できます。また、Studio をエンタープライズ管理システムに組み込むことも可能です。

## VeloDB Studio Desktop バージョン
### なぜデスクトップアプリケーションを提供するのか？
これまで、VeloDB Enterprise Manager、VeloDB Cloud、Alibaba Cloud で Studio WebUI の Web バージョンを提供してきました。

しかし、これらのバージョンはサーバーに配置するか、クラウドで完全にホストする必要があります。VeloDB カーネル向けに設計されており、管理システムアカウントへのログイン、支払い、複雑なネットワーク権限、更新のための管理者権限などが必要で、エンタープライズユーザーにより適しています。これらの設計は、一般ユーザーに多くの不便をもたらします。

Apache Doris ユーザーの利便性を図るため、Apache Doris 専用に設計・開発された GUI である VeloDB Studio Desktop バージョンを提供しています。主な利点は以下の通りです：

### VeloDB Studio Desktop Edition の特徴

:::info
Mac バージョンは 64 ビット macOS バージョン 13.0（Ventura）以上のシステムバージョンのみサポートします
:::
:::info
Windows バージョンは 64 ビット Windows 10 以上のシステムバージョンのみサポートし、Windows8、8.1、Windows Server 2012 はサポートされません
:::

**1. サーバー配置不要**

- Studio を単独で配置するためのサーバーを見つける必要がありません。VeloDB Studio Desktop インストールパッケージをダウンロードして、ダブルクリックするだけで使用できます。
- 他のアカウントにログインする必要がなく、アプリを開いて接続情報を入力するだけで Doris データベースに接続できます。

**2. 完全無料**

- 他のバージョンとは異なり、VeloDB Studio Desktop は永続的に無料で、ライセンス購入や支払いは不要です。

**3. Apache Doris 向けの設計**

- 他のバージョンの Studio は VeloDB カーネル上で動作するため、Apache Doris は使用できないか、互換性が限定されます。

- VeloDB Studio Desktop は Apache Doris 向けに設計され、Apache Doris と Apache Doris から派生した互換データベースをサポートします。

**4. より良いユーザーエクスペリエンス**

- より便利：デスクトップアプリケーションはコンピュータ上にあり、ブラウザを開いたり、アドレスを入力したり、アカウントにログインしたりする必要がなく、アイコンをダブルクリックするだけで開けます。

- より効率的：デスクトップはより強力なショートカットキーシステムとよりスムーズなウィンドウ管理をサポートします。接続はコンピュータに保存され、複数の接続を保存して、毎回接続情報を入力する必要がありません。

**5. Navicat と DBeaver に代わるネイティブツール**
- より強力な管理機能：Navicat や DBeaver などのよりクエリ重視のツールとは異なり、VeloDB Studio は セッション管理、ログ検索、権限管理、クエリ監査など、Apache Doris のより多くの機能をサポートします。

- より良いユーザーサポートと対応：VeloDB Studio チームは、機能リクエストや問題フィードバックにより迅速に対応し、Apache Doris に基づく新機能をより早く提供できます。

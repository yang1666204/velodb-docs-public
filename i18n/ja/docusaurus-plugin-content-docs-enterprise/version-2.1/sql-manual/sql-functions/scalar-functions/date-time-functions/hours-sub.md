## 説明

入力されたdatetime値から指定された時間数を減算した結果の新しいdatetime値を返します。

## 構文

```sql
HOURS_SUB(<date>, <hours>)
```
## パラメータ

| Parameter | デスクリプション                                      |
|-----------|--------------------------------------------------|
| `<date>`      | 入力する日時の値。DATETIME型またはDATE型を指定可能 |
| `<hours>`     | 減算する時間数。INT型 |

## Return Value

指定された時間数を入力日時から減算した後の時刻値を表すDATETIME型の値を返します。

## Example

```sql
SELECT HOURS_SUB('2020-02-02 02:02:02', 1);
```
```text
+------------------------------------------------------------+
| hours_sub(cast('2020-02-02 02:02:02' as DATETIMEV2(0)), 1) |
+------------------------------------------------------------+
| 2020-02-02 01:02:02                                        |
+------------------------------------------------------------+
```

## デスクリプション

入力されたdatetimeに指定した年数を加算した結果の新しいdatetime値を返します。

## Syntax

```sql
YEARS_ADD(<date>, <years>)
```
## パラメータ

| Parameter | デスクリプション                                      |
|-----------|--------------------------------------------------|
| `<date>`      | 入力する日時値で、DATETIME型またはDATE型を指定可能 |
| `<years>`     | 追加する年数、INT型         |

## Return Value

入力された`<date>`と同じ型（DATETIMEまたはDATE）の値を返します。これは入力された日時に指定された年数を追加した後の時刻値を表します。

## Example

```sql
SELECT YEARS_ADD('2020-01-31 02:02:02', 1);
```
```text
+-------------------------------------+
| years_add('2020-01-31 02:02:02', 1) |
+-------------------------------------+
| 2021-01-31 02:02:02                 |
+-------------------------------------+
```

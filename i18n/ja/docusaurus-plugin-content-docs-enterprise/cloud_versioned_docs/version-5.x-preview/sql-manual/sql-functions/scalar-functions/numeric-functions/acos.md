## 説明

`x`の逆余弦を返します。`x`が`-1`から`1`の範囲にない場合は`NULL`を返します。

## 構文

```sql
ACOS(<x>)
```
## パラメータ

| パラメータ | 説明 |
| -- | -- |
| `<x>` | アークコサインを計算する値 |

## 戻り値

パラメータ`x`のアークコサイン値をラジアン単位で表現したもの。

## 特殊なケース

- `x`が1の場合、0を返す
- `x`が0の場合、π/2を返す
- `x`が-1の場合、πを返す
- `x`が範囲[-1, 1]にない場合、`NULL`を返す
- `x`がNaNの場合、NaNを返す
- `x`が正または負の無限大の場合、`NULL`を返す
- `x`がNULLの場合、NULLを返す

## 例

```sql
select acos(1);
```
```text
+-----------+
| acos(1.0) |
+-----------+
|         0 |
+-----------+
```
```sql
select acos(0);
```
```text
+--------------------+
| acos(0.0)          |
+--------------------+
| 1.5707963267948966 |
+--------------------+
```
```sql
select acos(-1);
```
```text
+--------------------+
| acos(-1.0)         |
+--------------------+
| 3.141592653589793  |
+--------------------+
```
```sql
select acos(-2);
```
```text
+------------+
| acos(-2.0) |
+------------+
|       NULL |
+------------+
```
```sql
select acos(1.0000001);
```
```text
+-----------------+
| acos(1.0000001) |
+-----------------+
|            NULL |
+-----------------+
```
```sql
select acos(cast('nan' as double));
```
```text
+---------------------------+
| acos(cast('nan' AS DOUBLE)) |
+---------------------------+
| NaN                       |
+---------------------------+
```
```sql
select acos(cast('inf' as double));
```
```text
+---------------------------+
| acos(cast('inf' AS DOUBLE)) |
+---------------------------+
| NULL                      |
+---------------------------+
```

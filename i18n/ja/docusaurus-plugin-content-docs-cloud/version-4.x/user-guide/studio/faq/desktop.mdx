# Desktop Version FAQ

### クラスターに接続できず、`Login failed, please check the FE connection address or network connection`のプロンプトが表示される

1. FE接続アドレスとJDBCポートが正しいかどうかを確認してください。クラウドデータベースに接続している場合は、クラウドデータベースのネットワークポリシー（セキュリティグループ）が接続を許可しているかどうかを確認してください。
2. ネットワーク接続が正常であることを確認してください。`ping`コマンドを通じてネットワークが通じているかどうかを確認できます。
3. コンピュータ上のネットワークエージェント（Clashなど）のリクエストによって、クラスターへの接続が失敗する場合があります。エージェントを閉じる/コンピュータを再起動してから再接続を試すことができます。

### 使用中にエラーが発生し、`Internal Server Error`が表示される

1. ページの再読み込みを試してください（`CMD+R`）。
2. 終了をクリックしてクラスターに再接続してください。
3. メモリと接続管理の問題により接続プールが閉じられる可能性があるため、アプリケーションを再起動して再度操作を試してください。

### インターフェースが固まって操作できない

1. ページの再読み込みを試してください（`CMD+R`）。
2. アプリケーションを再起動して再度操作を試してください。

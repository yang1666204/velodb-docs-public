## 説明

指定された日付式に対応する曜日名を計算します。

## 構文

```sql
DAYNAME(<dt>)
```
## パラメータ

| Parameter | デスクリプション |
| -- | -- |
| `<dt>` | 計算される日付式 |

## Return Value

指定された日付式に対応する曜日の名前を返します。

## Examples

```sql
select dayname('2007-02-03 00:00:00');
```
```text
+--------------------------------+
| dayname('2007-02-03 00:00:00') |
+--------------------------------+
| Saturday                       |
+--------------------------------+
```

## 説明

`unhex`関数は、16進数文字列を元の文字列に変換するために使用されます。2つの16進数文字ごとに1バイトに変換します。無効な値がパラメータとして渡された場合、空文字列を返します。

## 構文

```sql
UNHEX(<str>)
```
## Parameters

| Parameter | Description |
| -- | -- |
| `<str>` | 16進数の文字列 |

## Return Value

入力文字列の長さが0または奇数の場合、`unhex`関数は空文字列を返します。
文字列が[0-9]、[a-f]、または[A-F]以外の文字を含む場合、`unhex`関数は空文字列を返します。
その他の場合、2文字ずつが16進数表現に変換され、文字列として連結されて出力されます。

## Examples

```sql
select unhex('@');
```
```text
+------------+
| unhex('@') |
+------------+
|            |
+------------+
```
```sql
select unhex('41');
```
```text
+-------------+
| unhex('41') |
+-------------+
| A           |
+-------------+
```
```sql
select unhex('4142');
```
```text
+---------------+
| unhex('4142') |
+---------------+
| AB            |
+---------------+
```

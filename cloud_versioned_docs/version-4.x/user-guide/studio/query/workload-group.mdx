# Workload Group Management
:::info
Workload Group Management supports Apache Doris 2.1.0 /VeloDB Cloud 4.0.0 and above.
:::

## Workload Group List
Workload Group Management supports the rapid creation, editing and viewing of Workload Group. Using Workload Group, you can manage the CPU/memory/IO resource usage used by querying and importing loads in the cluster, and control the maximum concurrency of queries in the cluster.

<img className='pt-5' src={`/img/studio/workload.png`} alt="workload" width={1400} height={600}/>

You can view more items in the table filter above the Workload Group list.
<img className='pt-5' src={`/img/studio/workload-more.png`} alt="workload-more" width={1400} height={600}/>

## Create a new Workload Group
In the New Workload Group interface, you can click on the question mark of the parameter, and the description of the parameter will be displayed.

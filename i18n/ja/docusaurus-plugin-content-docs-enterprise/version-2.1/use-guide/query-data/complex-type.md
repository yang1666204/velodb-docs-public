Dorisは、Array、Map、Struct、JSONなどの複合型をサポートしています。

Dorisは、これらの複合型専用に設計された様々な関数を提供しています。

サポートされている関数の詳細については、SQLマニュアル - Array functions、SQLマニュアル - Map functions data type、SQLマニュアル - Struct functions、およびSQLマニュアル - Json functionsを参照してください。

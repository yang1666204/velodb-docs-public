# 概要

VeloDB Cloudは、複数のクラウド上で動作するクラウドネイティブデータウェアハウスであり、一貫したユーザーエクスペリエンスと完全管理サービスを提供します。非常に高速で、コスト効率的、単一統合型、そして使いやすいという特徴があります。

このトピックでは、VeloDB Cloudコンソールに含まれる主要機能の概要とその操作方法について説明します。後続のトピックでは、各機能の詳細について説明します。

## 主要機能

-   **登録およびログイン**

-   **Warehouse管理**: 無料トライアル、有料warehouse作成、warehouseリストなどを提供します。

-   **クラスター管理**: ワンクリック作成、弾力的なリサイジング、高速アップグレード、削除などを提供します。

-   **接続**: プライベートネットワーク（VPC）およびパブリックネットワークでのwarehouse接続方法を提供します。パブリックネットワーク接続はホワイトリストをサポートします。

-   **メトリクス**: リソース使用量、クエリ、書き込みなどの観点でメトリクスを提供し、柔軟で使いやすいアラーム機能をサポートします。

-   **課金センター**: 組織およびwarehouse内部パーツの使用統計を提供します。課金は使用統計に基づいて行われます。

-   **その他**: 組織管理、アクセス制御、通知などを含みます。

## VeloDB Cloudのナビゲーション

VeloDB Cloudコンソールのwebインターフェースの全体的なレイアウトは以下の通りです：

![multi clusters](/images/cloud/multi-clusters.png)

### ナビゲーションバー

webインターフェースの左側に位置する**ナビゲーションバー**は、VeloDB Cloudの最も重要な概念である**Warehouse**の主要機能を提供します。これには、クラスター管理、接続、クエリ、メトリクス、使用統計などが含まれます。

### Warehouseセレクター

左側ナビゲーションバーの上部に位置する**Warehouseセレクター**は、現在の組織下にあるすべてのwarehouseを表示します。warehouseの切り替え、warehouse情報の表示、新しいwarehouseの作成などができます。

warehouseに切り替えた後、左側ナビゲーションバーのすべての機能を使用してそれを体験できます。

![warehouse-details](/images/cloud/warehouse-details.png)

### ユーザーメニュー

左側ナビゲーションバーの下部に位置する**ユーザーメニュー**は、ユーザーと組織に関連するいくつかの管理機能を提供します。これには、セキュリティ、通知、ユーザーと役割、課金などが含まれます。

![user-menu](/images/cloud/user-menu.png)

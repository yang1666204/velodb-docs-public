## デスクリプション

`x`の逆余弦を返します。`x`が`-1`から`1`の範囲内にない場合は`NULL`を返します。

## Syntax

```sql
ACOS(<x>)
```
## パラメータ

| Parameter | デスクリプション |
| -- | -- |
| `<x>` | acos値を計算する値 |

## Return Value

パラメータ`x`のacos値。

## Example

```sql
select acos(1);
```
```text
+-----------+
| acos(1.0) |
+-----------+
|         0 |
+-----------+
```
```sql
select acos(0);
```
```text
+--------------------+
| acos(0.0)          |
+--------------------+
| 1.5707963267948966 |
+--------------------+
```
```sql
select acos(-2);
```
```text
+------------+
| acos(-2.0) |
+------------+
|        nan |
+------------+
```

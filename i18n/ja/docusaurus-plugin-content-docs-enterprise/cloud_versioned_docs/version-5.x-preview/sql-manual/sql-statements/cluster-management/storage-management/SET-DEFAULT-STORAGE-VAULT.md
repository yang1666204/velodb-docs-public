## 説明

このステートメントは、Dorisでデフォルトストレージボルトを設定するために使用されます。デフォルトストレージボルトは、内部またはシステムテーブルのデータを格納するために使用されます。デフォルトストレージボルトが設定されていない場合、Dorisは正常に動作できません。デフォルトストレージボルトが一度設定されると、それを削除することはできません。


## 構文

```sql
SET <vault_name> AS DEFAULT STORAGE VAULT
```
## 必須パラメータ

| パラメータ名          | 説明                                                         |
|-------------------|--------------------------------------------------------------|
| `<vault_name>`    | ストレージボルトの名前。デフォルトストレージボルトとして設定したいボルトの一意識別子です。           |

## 使用上の注意:
> 1. ADMINユーザーのみがデフォルトストレージボルトを設定できます。

## 例

1. s3_vaultという名前のストレージボルトをデフォルトストレージボルトとして設定する。

   ```sql
   SET s3_vault AS DEFAULT STORAGE VAULT;
   ```

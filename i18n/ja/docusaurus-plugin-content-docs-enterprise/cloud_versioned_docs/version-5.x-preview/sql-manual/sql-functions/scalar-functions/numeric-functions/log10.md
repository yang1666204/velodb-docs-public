## Description

`x` の常用対数（底が `10` の対数）を返します。

## Alias

- DLOG10

## Syntax

```sql
LOG10(<x>)
```
## Parameters

| Parameter | Description |
|-----------|------------|
| `<x>`   | 真数は0より大きい値である必要があります |

## Return value

浮動小数点数を返します。特殊なケース：

- x が NULL の場合、`NULL` を返します
- x が NaN の場合、NaN を返します

## Example

```sql
select log10(1);
```
```text
+--------------------------+
| log10(cast(1 as DOUBLE)) |
+--------------------------+
|                      0.0 |
+--------------------------+
```
```sql
select log10(10);
```
```text
+---------------------------+
| log10(cast(10 as DOUBLE)) |
+---------------------------+
|                       1.0 |
+---------------------------+
```
```sql
select log10(16);
```
```text
+---------------------------+
| log10(cast(16 as DOUBLE)) |
+---------------------------+
|        1.2041199826559248 |
+---------------------------+
```
```sql
select log10(NULL);
```
```text
+-------------+
| log10(NULL) |
+-------------+
|        NULL |
+-------------+
```
```sql
select log10(cast('nan' as double));
```
```text
+------------------------------+
| log10(cast('nan' as double)) |
+------------------------------+
|                          NaN |
+------------------------------+
```

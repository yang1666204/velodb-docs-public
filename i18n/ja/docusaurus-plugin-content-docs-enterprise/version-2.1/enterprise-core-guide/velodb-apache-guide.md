# Apache Doris ドキュメント

VeloDB Enterprise 2.1は、Apache Doris 2.1をベースとした安定版ディストリビューションで、長期サポートを提供します。特定のカーネルに関する詳細なドキュメントについては、[Apache Doris 2.1](https://doris.apache.org/docs/2.1/gettingStarted/what-is-apache-doris)を参照してください。

VeloDB Enterprise 3.0は、Apache Doris 3.0をベースとした安定版ディストリビューションで、長期サポートを提供します。特定のカーネルに関する詳細なドキュメントについては、[Apache Doris 3.0](https://doris.apache.org/docs/3.x/gettingStarted/what-is-apache-doris)を参照してください。

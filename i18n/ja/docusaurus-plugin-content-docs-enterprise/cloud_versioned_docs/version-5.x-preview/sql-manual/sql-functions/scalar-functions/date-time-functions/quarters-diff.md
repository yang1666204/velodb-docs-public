## 説明

`QUARTERS_DIFF`関数は、2つの日付間の四半期数を計算します。この関数は2つの日付パラメータを受け取り、第1日付から第2日付を減算した後の四半期の差を返します。四半期の差は、月の差を3で割った値（ゼロに向かって丸められた値）と等価です。

## 構文

```sql
QUARTERS_DIFF(<enddate>, <startdate>)
```
## パラメータ

| Parameter     | Description                                                                           |
|---------------|--------------------------------------------------------------------------------------|
| `<enddate>`   | 終了日。差分を計算する際のより遅い日付を表します。`DATE`または`DATETIME`型をサポートします |
| `<startdate>` | 開始日。差分を計算する際のより早い日付を表します。`DATE`または`DATETIME`型をサポートします |

## 戻り値

`<enddate>`から`<startdate>`を引いて得られる四半期数を返します
- `<enddate>`または`<startdate>`のいずれかがNULLの場合、NULLを返します

## 例

```sql
select QUARTERS_DIFF('2021-03-25', '2020-10-25'), QUARTERS_DIFF('2020-10-25 10:00:00', '2022-12-25 11:00:00');
```
```text
+------------------------------------------+------------------------------------------------------------+
| QUARTERS_DIFF('2021-03-25','2020-10-25') | QUARTERS_DIFF('2020-10-25 10:00:00','2022-12-25 11:00:00') |
+------------------------------------------+------------------------------------------------------------+
|                                        1 |                                                         -8 |
+------------------------------------------+------------------------------------------------------------+
```

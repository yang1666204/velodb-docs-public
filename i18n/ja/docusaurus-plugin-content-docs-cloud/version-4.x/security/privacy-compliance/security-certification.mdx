# セキュリティ認証

VeloDBは専門的なセキュリティチームと法務チームを擁し、各主要国のデータプライバシー規制を厳格に遵守し、国内外のセキュリティコンプライアンス標準に積極的にベンチマークを行い、データセキュリティとコンプライアンスを確保しています。

VeloDBは現在、SoC2 Type1、Type2認証を取得しています。

**SoC2 Type1,Type2**

<img src={require("/images/cloud/security/privacy-compliance/soc.png").default} alt="" width="300" height="300"/>

System and Organization Controls (SOC) for Service Organizationsは、American Institute of Certified Public Accountants (AICPA)によって作成された内部統制レポートです。これらはサービス組織によって提供されるサービスを検査することを目的としており、エンドユーザーがアウトソーシングされたサービスに関連するリスクを評価し対処できるようにしています。VeloDBはSOC2 type2認証を取得しています。

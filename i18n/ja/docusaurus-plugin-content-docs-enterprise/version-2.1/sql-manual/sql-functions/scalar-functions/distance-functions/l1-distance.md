## 説明

L1空間における2点間の距離を計算します（ベクトルの値は座標です）

## 構文

```sql
L1_DISTANCE(<array1>, <array2>)
```
## パラメータ

| パラメータ | 説明 |
| -- |--|
| `<array1>` | 最初のベクトル（ベクトル値は座標）。入力配列のサブタイプは TINYINT、SMALLINT、INT、BIGINT、LARGEINT、FLOAT、DOUBLE です。要素数は array2 と一致している必要があります |
| `<array2>` | 2番目のベクトル（ベクトル値は座標）。入力配列のサブタイプは TINYINT、SMALLINT、INT、BIGINT、LARGEINT、FLOAT、DOUBLE をサポートします。要素数は array1 と一致している必要があります |

## 戻り値

L1空間における2つの点（ベクトル値は座標）の間の距離を返します。入力配列がNULLの場合、または配列内の任意の要素がNULLの場合、NULLが返されます。

## 例

```sql
SELECT L1_DISTANCE([4, 5], [6, 8]),L1_DISTANCE([3, 6], [4, 5]);
```
```text
+-----------------------------+-----------------------------+
| l1_distance([4, 5], [6, 8]) | l1_distance([3, 6], [4, 5]) |
+-----------------------------+-----------------------------+
|                           5 |                           2 |
+-----------------------------+-----------------------------+
```

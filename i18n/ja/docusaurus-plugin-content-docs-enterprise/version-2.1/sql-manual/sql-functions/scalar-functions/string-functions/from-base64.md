## 概要

入力文字列をBase64デコードした結果を返します。特殊なケース：

- 入力文字列が正しくない場合（Base64エンコード後にはあり得ない文字列が現れた場合）、NULLが返されます

## 構文

```sql
FROM_BASE64 ( <str> )
```
## パラメータ

| パラメータ | デスクリプション |
|------------|-----------------|
| `<str>`    | Base64デコードする文字列 |

## Return value

Parameter <str> Base64デコードの結果。特別なケース：

- 入力文字列が正しくない場合（Base64エンコード後には不可能な文字列が現れた場合）、NULLが返されます。

## Example

```sql
SELECT FROM_BASE64('MQ=='),FROM_BASE64('MjM0'),FROM_BASE64(NULL)
```
```text
+---------------------+---------------------+-------------------+
| from_base64('MQ==') | from_base64('MjM0') | from_base64(NULL) |
+---------------------+---------------------+-------------------+
| 1                   | 234                 | NULL              |
+---------------------+---------------------+-------------------+
```

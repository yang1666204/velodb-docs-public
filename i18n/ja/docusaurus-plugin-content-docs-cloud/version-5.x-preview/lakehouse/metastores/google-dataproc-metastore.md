この文書では、`CREATE CATALOG`ステートメントを通じてGoogle Dataproc Metastoreに接続およびアクセスする際にサポートされるパラメータについて説明します。

| Property Name        | Description                                                                                                     | Default Value | Required |
| -------------------- | --------------------------------------------------------------------------------------------------------------- | ------------- | -------- |
| `hive.metastore.uri` | Metastoreのアドレス。[Dataproc Metastore Services](https://console.cloud.google.com/dataproc/metastore)ページから取得できます | None          | Yes |

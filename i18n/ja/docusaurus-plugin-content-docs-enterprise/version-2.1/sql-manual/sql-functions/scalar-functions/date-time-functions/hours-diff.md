## Description

開始時刻と終了時刻の間の時間差を時間単位で計算します。

## Syntax

```sql
HOURS_DIFF(<end_date>, <start_date>)
```
## Parameters

| Parameter  | Description                                     |
|------------|-------------------------------------------------|
| `<end_date>`    | 終了時刻。DATETIME型またはDATE型を指定できます |
| `<start_date>`  | 開始時刻。DATETIME型またはDATE型を指定できます |

## Return Value

開始時刻と終了時刻の間の時間数を表すINT型を返します。

## Example

```sql
SELECT HOURS_DIFF('2020-12-25 22:00:00', '2020-12-25 21:00:00');
```
```text
+--------------------------------------------------------------------------------------------------------+
| hours_diff(cast('2020-12-25 22:00:00' as DATETIMEV2(0)), cast('2020-12-25 21:00:00' as DATETIMEV2(0))) |
+--------------------------------------------------------------------------------------------------------+
|                                                                                                      1 |
+--------------------------------------------------------------------------------------------------------+
```

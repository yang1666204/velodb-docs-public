### VeloDB Manager 24.0.5

2024年9月27日

バグ修正

* データベースアップグレードまたはFE再起動時のManagerの検出ロジックを最適化し、FE用の時刻同期サービスチェックを追加。
* アップグレード前のディレクトリ権限チェック問題を修正。
* スケーリング時のデプロイディレクトリがFEで終わらない場合に設定ファイルが見つからない問題を修正。
* 管理クラスタでアップグレード時にJDKが見つからない問題を修正。
* kill -15シグナル送信後にBEプロセスが自動再起動しない問題を修正。
* AgentがBEプロセス死亡を誤検知してstart scriptを実行する問題を修正。
* Manager内の管理操作後に一部のノードステータスが不整合になる問題を修正。
* File cache設定の不正な単位設定を修正。
* ダウンサイジング失敗後、設定変更で不明ステータス表示されるが概要ページで「実行中」表示される問題を修正。
* Mixノードが1つしかない場合でもダウンサイジングが許可される問題を修正。
* File cacheパスやストレージパスがBEの他のパスに占有されている、または重複設定されていることによるデプロイ失敗を最適化。
* 管理クラスタでのパラメータ変更が失敗する問題を修正。
* アラートプロキシが効果を発揮しない問題を修正。
* Agent登録時のエラーメッセージ表示不完全を修正。
* Agent戦略設定の自動適用失敗を修正。
* アップグレード時の不正確な容量検出メッセージを最適化。
* 新しく作成されたクラスタのステータスが不正（クラスタ作成失敗だがタスクステータスが「進行中」）な問題を修正。
* データTableのダーティデータがタスク異常を引き起こす問題を修正。
* Spring Webバージョンを5.3.33にアップグレード。
* フロントエンドでのクラスタ作成時の外部Tableキャッシュデータ最適化問題を修正。
* 各種フロントエンド最適化（ポート検証・制限追加、クラスタ整合性など）を実装。

### VeloDB Manager 24.0.3

2024年7月17日

バグ修正

* Doris Core 3.0.x（compute-storage結合モード）デプロイをサポート。
* ManagerとCoreバージョン間のバージョン関係制限を追加。
* Query Statisticsアラート戦略作成問題を修正。
* アップグレードディレクトリチェック時の不正なBEストレージパス認識問題を修正。
* agent自動アップグレード失敗問題を修正。
* クラスタとノードの無効ステータスプロンプトを最適化。
* アップグレードチェックエラーメッセージの不完全表示問題を修正。

### VeloDB Manager 24.0.2

2024年6月14日

バグ修正

* managerホスティング時の不正なプロセス情報取得問題を修正。
* 不整合なDorisログディレクトリによる不完全なログ表示問題を解決。
* 再起動中ノードのステータスが「再起動中」のまま残る問題を修正。
* 不正なFE/BE Not Aliveアラート式を修正。
* ホストインターフェースでのIO Util不正表示値を修正。
* manager初期化ページの繰り返し再入問題を解決。
* transparent hugepagesが無効化されているかのチェックに関するテキストを修正。
* クラスタアップグレードロジックを最適化し、事前チェックをアップグ

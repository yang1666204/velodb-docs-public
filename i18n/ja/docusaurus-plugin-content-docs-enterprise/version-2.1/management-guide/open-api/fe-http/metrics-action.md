# Metrics Action

## Request

`GET /api/metrics`

## デスクリプション

dorisのメトリクス情報を取得するために使用されます。

## Path parameters

なし

## Query parameters

* `type`

    オプションパラメータ。デフォルトですべてのメトリクスを表示し、以下の値を指定できます：
    - `core` コアメトリクスを出力
    - `json` メトリクスをjson形式で出力

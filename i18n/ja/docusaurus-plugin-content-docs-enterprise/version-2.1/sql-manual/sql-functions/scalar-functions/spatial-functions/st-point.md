## Description

与えられたX座標値に対して、Y座標値は対応するPointを返します。

現在の値は球面セット内でのみ意味を持ち、X/Yは経度/緯度に対応します；

## Syntax

```sql
ST_POINT( <x>, <y>)
```
## パラメータ

| パラメータ | 説明 |
|-----|--------------|
| `<x>` | x座標 |
| `<y>` | y座標 |

## 戻り値

指定された水平座標と垂直座標に対応する位置情報

## 例

```sql
SELECT ST_AsText(ST_Point(24.7, 56.7));
```
```text
+---------------------------------+
| st_astext(st_point(24.7, 56.7)) |
+---------------------------------+
| POINT (24.7 56.7)               |
+---------------------------------+
```

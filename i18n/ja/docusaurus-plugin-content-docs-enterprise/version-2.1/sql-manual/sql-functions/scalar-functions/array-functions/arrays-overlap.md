## デスクリプション

左の配列と右の配列に共通する要素が含まれているかどうかを判定します

## Syntax

```sql
ARRAYS_OVERLAP(<left>, <right>)
```
## パラメータ

| Parameter | デスクリプション |
|--|--|
| `<left>` | 判定対象の配列 |
| `<right>` | 判定対象の配列 |

## Return Value

判定結果を返します：1：左右の配列に共通要素がある；0：左右の配列に共通要素がない；NULL：左または右の配列がNULL、または左右の配列のいずれかの要素がNULL

## Example

```sql
SELECT ARRAYS_OVERLAP(['a', 'b', 'c'], [1, 2, 'b']);
```
```text
+--------------------------------------------------+
| arrays_overlap(['a', 'b', 'c'], ['1', '2', 'b']) |
+--------------------------------------------------+
|                                                1 |
+--------------------------------------------------+
```

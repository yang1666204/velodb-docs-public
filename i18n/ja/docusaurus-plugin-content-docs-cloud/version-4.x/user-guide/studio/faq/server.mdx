# サーバー Version FAQ

## VeloDB Cloud Studio

### ログインに失敗し、ログインできない

1. ユーザー名とパスワードが正しいかどうかを確認してください。テストにはmysql cliを使用できます。
2. ネットワーク接続が正常かどうかを確認してください。`ping`コマンドを使用してネットワークが通じているかどうかを確認できます。

### VeloDB Cloudコンソールに障害が発生した場合、Studioは正常に使用できますか？

はい、コンソールが停止していても、Studioは正常に動作しています。Studioとコンソールはそれぞれ独立したサービスです。

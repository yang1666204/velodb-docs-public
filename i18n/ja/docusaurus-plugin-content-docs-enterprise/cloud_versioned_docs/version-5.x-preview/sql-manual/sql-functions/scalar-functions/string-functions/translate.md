## 説明

TRANSLATE関数は文字単位での文字列置換を実行し、マッピングルールに従ってソース文字列内の文字を変換します。この関数は、ソース文字列内の各文字について、'from'文字列に含まれる文字を'to'文字列の同じ位置にある対応する文字に置き換えます。

## 構文

```sql
TRANSLATE(<source>, <from>, <to>)
```
## パラメータ
| Parameter | Description                                         |
| --------- | --------------------------------------------------- |
| `<source>` | 変換対象のソース文字列。型: VARCHAR    |
| `<from>` | 置換される文字のセット。型: VARCHAR |
| `<to>` | 置換文字のセット。型: VARCHAR    |

## 戻り値

文字マッピングルールに従って変換された文字列を表すVARCHAR型を返します。

文字マッピングルール:
- 'from'と'to'の文字列内の位置に基づいて一対一の文字マッピングを確立します
- 'from'の1文字目は'to'の1文字目にマップされ、2文字目は2文字目に、といった具合になります
- 'from'に重複文字が含まれる場合、最初に出現したもののマッピングを使用し、その後の重複は無視されます
- ソース文字列内の'from'文字列に含まれていない文字は変更されません

特殊ケース:
- いずれかのパラメータがNULLの場合、NULLを返します
- ソースが空文字列の場合、空文字列を返します
- 'from'が空文字列の場合、元のソース文字列を返します
- 'to'が空文字列の場合、'from'に出現するソースのすべての文字を削除します
- 'to'文字列が'from'より短い場合、余分な'from'文字に対応するソース内の文字は削除されます

## 例

1. 基本的な文字置換

```sql
SELECT translate('abcd', 'a', 'z');
```
```text
+---------------------------+
| translate('abcd', 'a', 'z') |
+---------------------------+
| zbcd                      |
+---------------------------+
```
2. 同じ文字の複数回置換

```sql
SELECT translate('abcda', 'a', 'z');
```
```text
+----------------------------+
| translate('abcda', 'a', 'z') |
+----------------------------+
| zbcdz                      |
+----------------------------+
```
3. 特殊文字の置換

```sql
SELECT translate('Palhoça', 'ç', 'c');
```
```text
+--------------------------------+
| translate('Palhoça', 'ç', 'c') |
+--------------------------------+
| Palhoca                        |
+--------------------------------+
```
4. 文字の削除（空の'to'文字列）

```sql
SELECT translate('abcd', 'a', '');
```
```text
+----------------------------+
| translate('abcd', 'a', '') |
+----------------------------+
| bcd                        |
+----------------------------+
```
5. 'from'文字列内の重複文字（最初のマッピングのみを使用）

```sql
SELECT TRANSLATE('abacad', 'aac', 'zxy');
```
```text
+-----------------------------------+
| TRANSLATE('abacad', 'aac', 'zxy') |
+-----------------------------------+
| zbzyzd                            |
+-----------------------------------+
```
6. 'to'文字列が'from'より短い場合（余分な文字を削除）

```sql
SELECT TRANSLATE('abcde', 'ace', 'xy');
```
```text
+-------------------------------+
| TRANSLATE('abcde', 'ace', 'xy') |
+-------------------------------+
| xbyd                          |
+-------------------------------+
```
7. UTF-8文字置換

```sql
SELECT TRANSLATE('ṭṛì ḍḍumai', 'ṭṛ', 'ab');
```
```text
+-----------------------------------+
| TRANSLATE('ṭṛì ḍḍumai', 'ṭṛ', 'ab') |
+-----------------------------------+
| abì ḍḍumai                        |
+-----------------------------------+
```
8. 数値文字の置換

```sql
SELECT TRANSLATE('a1b2c3', '123', 'xyz');
```
```text
+----------------------------------+
| TRANSLATE('a1b2c3', '123', 'xyz') |
+----------------------------------+
| axbycz                           |
+----------------------------------+
```
9. 特殊記号の置換

```sql
SELECT TRANSLATE('hello@world.com', '@.', '-_');
```
```text
+--------------------------------------------+
| TRANSLATE('hello@world.com', '@.', '-_')   |
+--------------------------------------------+
| hello-world_com                            |
+--------------------------------------------+
```

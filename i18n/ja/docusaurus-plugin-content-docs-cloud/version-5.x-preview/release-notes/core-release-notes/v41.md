# v4.1

この記事では、VeloDB Core v4.1のリリースノートについて説明します。

> **注記** VeloDB Core v4.1は[Apache Doris v3.1](https://doris.apache.org/docs/releasenotes/v3.1/release-3.1.0)をベースに開発されました。


## v4.1.1 (December 10, 2025)

改善

- [Apache Doris v3.1.3](https://doris.apache.org/docs/releasenotes/v3.1/release-3.1.3)のすべての機能、改善、バグ修正をマージしました。

## v4.1.0 (October 17, 2025)

改善

- [Apache Doris v3.1.1](https://doris.apache.org/docs/releasenotes/v3.1/release-3.1.1)のすべての機能、改善、バグ修正をマージしました。

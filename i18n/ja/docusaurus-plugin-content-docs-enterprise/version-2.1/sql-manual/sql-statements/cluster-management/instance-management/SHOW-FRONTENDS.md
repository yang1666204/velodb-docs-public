## Description

このステートメントは、FEノードの基本ステータス情報を表示するために使用されます。

## Syntax

```sql
SHOW FRONTENDS
```
## 戻り値

| カラム                      | 備考                                                                                          |
|--------------------|-----------------------------------------------------------------------------------------------|
| Name               | Dorisにおける現在のFEの名前。この名前は通常`fe`で始まるランダム文字列です。 |
| Host               | 現在のFEのIPアドレスまたはホスト名。                                                |
| EditLogPort        | 現在のFEのbdbje通信ポート。                                               |
| HttpPort           | 現在のFEのhttp通信ポート。                                                                             |
| QueryPort          | 現在のFEのMySQLプロトコル通信ポート。                                                                          |
| RpcPort            | 現在のFEのthrift RPC通信ポート。                                                                       |
| ArrowFlightSqlPort | 現在のFEのArrowFlightプロトコル通信ポート。                                                                    |
| Role               | 現在のFEの役割。可能な値はFOLLOWERおよびOBSERVERです。                                                           |
| IsMaster           | 現在のFEがMasterとして選出されているかどうか。                                                                           |
| ClusterId          | 現在のDorisクラスタのID。通常はランダムに生成された数値です。                                                                  |
| Join               | 現在のFEノードが現在のDorisクラスタに正常に参加しているかどうかを示すために使用されます。                                                                 |
| Alive              | 現在のFEが生きているかどうか。                                                                                    |
| ReplayedJournalId  | 現在のFEが再生した最大のメタデータログのID。                                                                         |
| LastStartTime      | 現在のFEが開始されたタイムスタンプ。                                                                                  |
| LastHeartbeat      | 現在のFEが送信した最後の成功したハートビートのタイムスタンプ。                                                                           |
| IsHelper           | 現在のFEがbdbjeのヘルパーノードかどうか。                                                                  |
| ErrMsg             | 現在のFEのハートビートが失敗した時のエラーメッセージ。                                                                              |
| Version            | 現在のFEのバージョン情報。                                                                                   |
| CurrentConnected   | 現在のクライアント接続が現在のFEノードに接続されているかどうか。                                                                          |

## アクセス制御要件

このSQLを実行するユーザーは、少なくとも以下の権限を持っている必要があります：

| 権限  | オブジェクト | 備考 |
|------------|----|----|
| ADMIN_PRIV |    |    |

## 使用上の注意

クエリ結果をさらにフィルタリングする必要がある場合は、テーブル値関数[frontends()](../../../sql-functions/table-valued-functions/frontends.md)を使用できます。SHOW FRONTENDSは以下のステートメントと同等です：

```sql
SELECT * FROM FRONTENDS();
```

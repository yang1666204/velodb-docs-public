# Release Notes

:::info
VeloDB Studio Server and Desktop versions use common version update logs, but some features of the Server version may be updated earlier than or later than the release date.
:::

## v1.1.0
Release date: April 29, 2025

VeloDB Studio v1.1.0 is released! In this version, we released the Windows version and Mac Intel chip version, made major updates to the SQL console, as well as some bug fixes and feature optimizations.

The following are the specific updates:

### New Features

- SQL console supports multi-line SQL query

- Profile supports display in original Text format

- Support REFRESH CATALOG

- Support whitelisting to restrict some users' login (Server version, running on Cloud BYOC repository)

- User/role management added COMMENT

- SQL query results support replication to...

### Optimization
- SQL query results optimize performance under large data volume

- Studio no longer depends on MS services (Server version, running on Cloud SaaS repository)


### Bug Fixed

- Search and analyze the problem of unavailable Chinese search

- Load Management 0.0001 mem_limit error

- SQL Console Viewing Part View Error


## v1.0.0

Release date: March 27, 2025

The first version of VeloDB Studio Desktop is released! This is an important milestone and we are excited to share this version with you.

In this version, we bring you the following features:

- SQL console optimized for Apache Doris

- Provides search analysis for Apache Doris logging capabilities

- Visual session management and query audit

- Visual Profile Analysis

- Permission Management

- Data Panel

- Visual import task viewing

- Supports multi-connection and SSH tunneling (Desktop version only)

- Dark /Light Theme (Server version is soon supported)
We hope you will enjoy this version and hope you can provide feedback to help us improve by using this version.

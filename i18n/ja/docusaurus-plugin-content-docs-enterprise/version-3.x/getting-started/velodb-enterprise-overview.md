# イントロダクション

VeloDB Enterpriseは、Apache Dorisをベースとした商用製品です。オンプレミス、VM、またはK8sでの使用が推奨されており、コミュニティ版と比較してより良い運用・開発体験を実現し、セキュリティと安定性についてもより強力なサポートを提供します。

## 長期コアバージョン

オープンソースのコアバージョンは急速に反復し、ある程度の安定性を犠牲にして新機能や機能の体験をユーザーに提供しています。オープンソース版とは対照的に、エンタープライズエディションは長期サポート付きの安定したディストリビューションを提供し、本番環境での安定性を確保するために重要なバグの専用修正を日次ベースで提供します。

* **安定性**：コミュニティによる広範囲な使用と厳格なテストを通じて厳密にテストされた安定したコア。
* **Long-Term Support (LTS)**：12-24ヶ月のサポートを提供するLTSバージョンを提供し、バグ修正バージョンを定期的にリリースします。
* **互換性**：Dorisコミュニティエディションと**完全互換**であり、**シームレスな移行を可能**にします。コミュニティのアップデート後、VeloDBも対応するディストリビューションを提供し、最新の機能と最適化との整合性を確保します。

## 運用・開発ツール

直感的なインターフェースを通じてユーザーにシンプルで使いやすいビジュアル運用体験を提供するワンストップデータベースクラスター管理ツールを提供します。これにより運用効率が大幅に向上し、エラーや中断が削減され、サービス品質と安定性が改善されます。

* **クラスターデプロイとテイクオーバー**：オンプレミス、VM、K8sのいずれであっても、VeloDB Managerは簡単にDorisクラスターをデプロイし、テイクオーバーできます。複雑なコマンドライン操作はもう必要なく、すべて直感的なインターフェースを通じて実行できます。
* **柔軟なクラスター管理**：必要に応じてクラスターのスケーリング、再起動、アップグレード、パラメーター設定操作を実行します。これによりビジネスの柔軟性とシステムの可用性が大幅に向上します。
* **監視とアラート**：VeloDB Managerは事前定義された監視メトリクスのセットを備え、カスタマイズ可能なアラート通知をサポートします。ユーザーはニーズに応じてアラート閾値と通知方法を設定でき、クラスター運用に関する情報をタイムリーに受信し、早期警告と積極的な問題解決を可能にします。
* **便利なWebUI**：各クラスターにWebUIエントランスが提供され、ユーザーは簡単にクエリを実行し、データを表示し、権限を管理できます。

## 有償技術サポートサービス

Apache Dorisの創設チームによって構築されたVeloDBは、本番環境でのDoris使用に関する懸念に対応するための迅速で業界をリードする技術サポートを提供する能力があります。VeloDB Enterpriseは、お客様の特定のニーズに合わせたサービスレベル契約（SLA）に基づく専門的な有償技術サポートを提供します。このサポートには以下が含まれます：

* **リスク軽減**：定期的な検査を実施し、潜在的なシステムリスクを迅速に特定・排除します。
* **問題解決**：24時間365日の専用サポートを確保するための厳格なサービスレベル契約；重要なバグの専用修正を日次アップデートで提供します。
* **システム最適化**：製品使用トレーニング、業界ベストプラクティスの共有、お客様との協働最適化活動を通じてシステムパフォーマンスとコスト効率を向上させます。

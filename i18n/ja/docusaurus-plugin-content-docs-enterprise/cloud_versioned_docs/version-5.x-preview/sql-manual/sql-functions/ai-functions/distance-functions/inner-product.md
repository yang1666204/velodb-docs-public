## 説明

同じサイズの2つのベクトルのスカラー積を計算します

## 構文

```sql
INNER_PRODUCT(<array1>, <array2>)
```
## パラメータ

| パラメータ | 説明 |
| -- |--|
| `<array1>` | 最初のベクター。入力配列のサブタイプは次をサポートします：TINYINT、SMALLINT、INT、BIGINT、LARGEINT、FLOAT、DOUBLE。要素数はarray2と一致している必要があります |
| `<array1>` | 2番目のベクター。入力配列のサブタイプは次をサポートします：TINYINT、SMALLINT、INT、BIGINT、LARGEINT、FLOAT、DOUBLE。要素数はarray1と一致している必要があります |

## 戻り値

同じサイズの2つのベクターのスカラー積を返します。入力配列がNULLの場合、または配列内のいずれかの要素がNULLの場合、NULLが返されます。

## 例

```sql
SELECT INNER_PRODUCT([1, 2], [2, 3]),INNER_PRODUCT([3, 6], [4, 7]);
```
```text
+-------------------------------+-------------------------------+
| inner_product([1, 2], [2, 3]) | inner_product([3, 6], [4, 7]) |
+-------------------------------+-------------------------------+
|                             8 |                            54 |
+-------------------------------+-------------------------------+
```
